Features:
─────────
✓ Enable/disable GDB server with checkbox
✓ Configurable port (1024-65535)
✓ Pause on start option for waiting for debugger
✓ Dynamic info panel showing connection instructions
✓ Port value updates in real-time in info panel
✓ Controls disabled when GDB server is not enabled
✓ All settings persisted to configuration file


When GDB Server is Disabled:
────────────────────────────
- Port input is grayed out
- Pause on Start checkbox is grayed out  
- Information panel is hidden


Usage Flow:
───────────
1. Open Win32Emu GUI
2. Click Settings in navigation
3. Scroll to "GDB Server" section
4. Check "Enable GDB Server"
5. Adjust port if needed
6. Choose whether to pause on start
7. Launch any game from library
8. GDB server starts automatically
9. Connect from Ghidra/IDA/GDB using displayed instructions
